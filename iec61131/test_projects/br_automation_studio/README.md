# B&R Automation Studio Test Project

This directory will contain a complete B&R Automation Studio project for testing the IEC 61131-3 Protocol Buffers implementation.

## 🎯 Purpose

This test project serves multiple purposes:
- **Runtime Testing**: Validate the ProtoST library functions correctly
- **Integration Testing**: Test generated ST code compiles and runs
- **Example Implementation**: Show developers how to use Protocol Buffers in B&R
- **Regression Testing**: Ensure changes don't break existing functionality

## 🧪 Test Scenarios

### Basic Functionality Tests
1. **Simple Message Encoding**
   - Encode basic types (int, float, bool)
   - Verify output buffer contents
   - Check encoded size calculation

2. **Simple Message Decoding**
   - Decode known good protobuf data
   - Verify field values match expected
   - Test error handling for invalid data

### Advanced Feature Tests
3. **Optional Fields**
   - Encode messages with/without optional fields
   - Verify has_field flags work correctly
   - Test default value handling

4. **String Fields**
   - Encode/decode string data
   - Test string length limits
   - Handle UTF-8 encoding (if supported)

5. **Repeated Fields**
   - Test array-based repeated fields
   - Verify count and size handling
   - Test maximum array limits

### Error Handling Tests
6. **Buffer Overflow**
   - Test encoding to undersized buffers
   - Verify error reporting
   - Check graceful failure

7. **Invalid Data**
   - Test decoding corrupted protobuf data
   - Verify error messages
   - Test recovery behavior

### Performance Tests
8. **Large Messages**
   - Test maximum supported message sizes
   - Measure encoding/decoding performance
   - Test memory usage patterns

9. **Repeated Operations**
   - Test multiple encode/decode cycles
   - Check for memory leaks or corruption
   - Verify state handling
